## Overview

Sdk that inteacts with horizon. It uses `zagg-stellar-base` internally.

## Installation

`npm i zagg-stellar-sdk`

## Building

`npm i`


## Publishing on npm

- Make changes in code
- Update the version in package.json.
- `gulp build`
- `npm login`
- `npm version major | minor | patch --force`
- `npm publish`
